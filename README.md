# What is it?
It's a project for archiving something.

Such as entity document (like SCP foundation websites on wikidot) neither level document (like The Backrooms websites on wikidot) and more.

# Requirements
Just PHP and nginx.

# Quick make up

    git clone https://github.com/BL-BlueLighting/ArchiveRoom.git
    cd ArchiveRoom
    php manage.php user add <Your Account Name> <Password>
    php manage.php perm add <Your Account Name> Administrator

Then, open your website.

# How to write documents?
Just use markdown syntax.
